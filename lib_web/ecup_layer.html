마크업 검수 레이어

1. 마크업 검수 레이어 간단히 사용하기

1) 레이어 관리자 가져오기

2) 레이어 테마(디자인) 및 타입(노출 방식)의 설정

3) 개별버튼 또는 그룹버튼 생성

이렇게 3단계로 나뉘며 (기본 예시) 는 다음과 같다.

var layer = ecup.markupLayer();

// 레이어 설정하기

layer.set({

type: 'external',

theme: 'dark'

});

// 버튼 생성하기 (옵션으로 주는 방법도 있음)

layer({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 기능': function() {

// 버튼2 기능

}

});

2. 레이어 설정하기

1) type: 레이어 보기 방식 설정

- windows: 팝업창 형식으로 보기

- external: 디자인을 가리지 않게 창 내부에서 보기

- internal: 레이어 형식으로 보기

2) theme: 레이어 디자인 설정

- default, dark, naver

3) groupType: 버튼그룹의 동작방식 설정

- default: 설정하지 않는 경우 일반 버튼처럼 동작

- checkbox: 해당그룹 버튼들이 체크박스처럼 토글로 동작

- radio: 해당그룹 버튼들이 라디오처럼 동작

var layer = ecup.markupLayer();

layer.set({

type: 'windows', // windows, external, internal

theme: 'dark', // default, dark, naver

groupType: {

'그룹이름 1' : 'checkbox' // 해당 그룹의 버튼들이 체크박스 처럼 동작

'그룹이름 2' : 'radio' // 해당 그룹의 버튼들이 라디오 처럼 동작

}

});

3. 레이어 개별버튼 정의하기

그룹화 되지 않는 개별 버튼들을 생성

버튼의 동작을 정의하는 방법에는 두가지가 있는데

직접 함수로 구현하는 방법, 옵션을 주는 방법 2가지가 있음.

var layer = ecup.markupLayer();

// 버튼동작을 함수로 정의하기

layer({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': function() {

// 버튼2 기능

}

});

// 버튼동작을 옵션으로 정의하기

layer({

'버튼1 이름': {

target: '#div', // 옵션을 적용시킬 타깃의 셀렉터,

addClass: 'class1 class2 class3' // 타깃에게 부여할 옵션 1 (옵션의 종류 보기)

toggleAttr: 'aria-label="닫기"' // 타깃에게 부여할 옵션 2 .. (여러개 가능)

},

'버튼2 이름': ...

});

4. 레이어 그룹버튼 정의하기

버튼을 그룹화 시켜 정의할 수 있음.

그룹화의 장점으로는

1) 버튼을 그룹화 할 수 있다.

2) 버튼의 동작 방식을 설정할 수 있다.

- default: 기본 버튼 (기본값)

- checkbox: 버튼이 체크박스 처럼 동작 (해당 버튼의 기능이 체크박스처럼 토글 적용)

- radio: 버튼이 라디오버튼 처럼 동작 (같은 그룹 내에 특정 버튼을 누르면 그 외의 버튼들의 적용된 부분이 해제)

3) 옵션 형태로 줄 때, 공통 타깃을 정의할 수 있다.

var layer = ecup.markupLayer();

// 그룹화 하기

layer('그룹 이름').group({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': {

target: '#div',

addClass: 'class1 class2 class3'

}

});

// 동작 방식 설정하기

// layer.set 설정에서 한번에 설정도 가능하다. (layer.set)

layer('그룹 이름').setType('radio').group({

...

});

// 옵션의 공통 타깃을 정의 (group 함수의 마지막 인자가 버튼정의, 그 이전의 모든 인자가 공통 타깃이 됨)

layer('그룹 이름').group('#div', '.test', '#span', {

'버튼1 이름': function() {

// 일반 함수로 주면 공통 타깃과 무관하게 그냥 적용.

},

'버튼2 이름': {

// 타깃을 주지 않으면 옵션들이 공통 타깃에게 적용됨.

addClass: 'class1 class2 class3' // 즉, #div, .test, #span 에 적용.

},

'버튼3 이름': {

// 타깃을 옵션에서 새로 부여하면 공통타깃은 무시되고 현재 타깃에게 적용.

target: '#div',

hide: 400 // 즉, #div 에만 적용

}

});

5. 버튼 옵션의 종류보기

버튼의 옵션은 마크업 검수시 가장 자주 사용되는 9가지만 존재한다.

클래스 / 속성 의 경우 문자열로 값을 주고

보이기 / 숨기기 의 경우 속도를 숫자 값으로 준다.

옵션보기 복수 적용
클래스 추가 addClass: 'class1 class2' 공백
삭제 removeClass: 'class1 class2'
토글 toggleClass: 'class1 class2'
속성 추가 addAttr: 'role="button" aria-label="닫기"' 공백
쉼표
삭제 removeAttr: 'role, button'
토글 toggleAttr: 'role="button", aria-label="열기"'
Display 보기 show: 400 -
숨기기 hide: 0
토글 toggle: 0
