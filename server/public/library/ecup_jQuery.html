크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

  1. 1. 마크업 검수 레이어 생성
  2. 2. 자주 사용되는 UI 인터랙션을 간단한 jQuery 메서드로 구현
    • DOM 의 속성을 다루는 메서드
    • DOM 의 인라인 스타일을 다루는 메서드
  3. 3. 기타 생산성을 향상시켜줄 부가 기능들
마크업 검수 레이어 자주 사용되는 UI 인터랙션 기타 부가기능

자주 사용되는 UI 인터렉션 ( jQuery 확장 )

1. DOM 의 속성을 다루는 메서드

기존의 jQuery에서 .attr() 메서드를 통해 속성을 다뤘는데

이 기능을 좀 더 직관적이고 여러가지 속성을 한번에 다룰 수 있도록 확장시켰습니다.

1) 속성 추가하기

// 예시

$('#test').addAttr('role="button" aria-label="닫기"');

$('#test').addAttr('role="button"', 'aria-label="닫기"');

2) 속성 삭제하기

// 예시

$('#test').removeAttr('role, aria-label');

$('#test').removeAttr('role', 'aria-label');

3) 속성 토글하기

// 예시

$('#test').toggleAttr('role="button", aria-label="열기"');

$('#test').toggleAttr('role="button"', 'aria-label="열기"');

2. DOM 의 인라인 스타일을 다루는 메서드

기존의 jQuery에서 .css() 메서드를 통해 인라인스타일을 다뤘는데

이 기능을 좀 더 직관적이고 여러가지 스타일을 한번에 다룰 수 있도록 확장시켰습니다.

1) 인라인스타일 추가하기

$('#test').addStyle('background-color: #555; color: red;');

$('#test').addStyle('background-color: #555', 'color: red');

2) 인라인스타일 삭제하기

$('#test').removeStyle('color, font-size');

$('#test').removeStyle('color', 'font-size');

3) 인라인스타일 토글하기

$('#test').toggleStyle('width: 100%; height: 100%;');

$('#test').toggleStyle('width: 100%', 'height: 100%');

맨 위로 DOM 의 속성을 다루는 메소드 DOM 의 인라인 스타일을 다루는 메소드
