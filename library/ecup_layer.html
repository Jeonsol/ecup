NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 마크업검수 레이어 # 코멘트 # UI 컴포넌트
ecup.markupLayer.set( { ... } )
Options

Type: Obeject

  • type (default: external)
    Type: string

    레이어 보기 방식

    • external: 디자인을 가리지 않게 창 내부에서 보기

    • internal: 레이어 형식으로 보기

    • windows: 새창 형식으로 보기

  • theme (default: dark)
    Type: string

    레이어 디자인 색상 테마

    • default, dark, naver

  • buttonType (default: check)
    Type: string

    버튼그룹의 동작방식 설정

    • check: 그룹내 버튼들이 체크박스처럼 토글로 동작

    • button: 그룹내 버튼들이 일반 버튼처럼 동작

    • radio: 그룹내 버튼들이 라디오처럼 동작

Example

ecup.markupLayer.set({

type: 'windows', // windows, external, internal

theme: 'dark', // default, dark, naver

buttonType: {

'그룹이름 1' : 'check' // 해당 그룹의 버튼들이 체크박스 처럼 동작

'그룹이름 2' : 'radio' // 해당 그룹의 버튼들이 라디오 처럼 동작

}

});

ecup.markupLayer( { key : value, ... } )
Options

Type: Obeject

  • key : value
    Type
    - key : string
    - value : object or function

    단일 버튼 이벤트 제어

    • key: 버튼 이름 (버튼의 고유 값)

    • value: 버튼 이벤트 발생 시 동작할 내용

** value
ecup 내에서 제공하는 옵션을 사용하거나,
직접 함수를 할당하는 2가지 방법으로 사용할 수 있습니다.

Example

// 버튼동작을 함수로 정의하기

ecup.markupLayer({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': function() {

// 버튼2 기능

}

});

// 버튼동작을 옵션으로 정의하기

ecup.markupLayer({

'버튼1 이름': {

target: '#div', // 옵션을 적용시킬 타깃의 셀렉터,

addClass: 'class1 class2 class3' // 타깃에게 부여할 옵션 1 (옵션의 종류 보기)

toggleAttr: 'aria-label="닫기"' // 타깃에게 부여할 옵션 2 .. (여러개 가능)

},

'버튼2 이름': ...

});

ecup.markupLayer( name ).group( target, { key : value, ... } )
Name

Type: String

  • 버튼 그룹명

    • 그룹을 식별하는 고유한 키값 (언어 무관)

Group Options

Type: Obeject

  • target
    Type: String

    공동 target 정의

    • key: 버튼 이름 (버튼의 고유 값)

    • value: 버튼 이벤트 발생 시 동작할 내용

  • key : value
    Type
    - key : string
    - value : object or function

    버튼 이벤트 제어

    • key: 버튼 이름 (버튼의 고유 값)

    • value: 버튼 이벤트 발생 시 동작할 내용

** Group Options
▪ target : 그룹내의 버튼들의 여러 공동타깃을 한번에 정의 할 수 있습니다.
▪ key : value : ecup.markupLayer( { . . . } )에서의 와 사용 방법이 동일합니다.

** 단일 버튼과 달리 그룹 버튼은 버튼의 타입을 지정할 수 있습니다.

Example

// 그룹화 하기

ecup.markupLayer('그룹 이름').group({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': {

target: '#div',

addClass: 'class1 class2 class3'

}

});

// 동작 방식 설정하기

// ecup.markupLayer.set 설정에서 한번에 설정도 가능하다. (ecup.markupLayer.set)

ecup.markupLayer('그룹 이름').setType('radio').group({

...

});

// 옵션의 공통 타깃을 정의 (group 함수의 마지막 인자가 버튼정의, 그 이전의 모든 인자가 공통 타깃이 됨)

ecup.markupLayer('그룹 이름').group('#div', '.test', '#span', {

'버튼1 이름': function() {

// 일반 함수로 주면 공통 타깃과 무관하게 그냥 적용.

},

'버튼2 이름': {

// 타깃을 주지 않으면 옵션들이 공통 타깃에게 적용됨.

addClass: 'class1 class2 class3' // 즉, #div, .test, #span 에 적용.

},

'버튼3 이름': {

// 타깃을 옵션에서 새로 부여하면 공통타깃은 무시되고 현재 타깃에게 적용.

target: '#div',

hide: 400 // 즉, #div 에만 적용

}

});

Button Options
클래스

복수적용 : 공백

  • 추가

    addClass: 'class1 class2'

    삭제

    removeClass: 'class1 class2'

    토글

    toggleClass: 'class1 class2'

속성

복수적용 : 공백 or 쉼표(,)

  • 추가

    addAttr: 'role="button" aria-label="닫기"'

    삭제

    removeAttr: 'role, aria-label'

    토글

    toggleAttr: 'role="button", aria-label="열기"'

인라인 스타일

복수적용 : 쉼표(,)

  • 추가

    addStyle: 'font-size: 13px; color #333;'

    삭제

    removeStyle: 'font-size, color'

    토글

    toggleStyle: 'width: 100%; background: red;'

Display

  • 보이기

    show: 400

    숨기기

    hide: 0

    토글

    toggle: 0

** 마크업 디자인 검수시 자주 사용되는 동작을 미리 정의해 놓았습니다.

ecup.comment( target : comment ) UI 메모 남기기
Options

Type: String, String

  • target : comment

    UI 코멘트 표시

    • target: 코멘트가 표시될 dom

    • comment: 코멘트 내용

Property
  • show
    Type: boolean

    UI 코멘트 화면 표시 유무

    • true: 보이기

    • false: 숨기기

**
디자이너는 마크업개발자에게 제플린의 코멘트 기능을 활용해 기타 의견을 전달할 수 있지만,
마크업개발자는 디자이너에게 UI 검수를 받을 때 메모를 남길 수 없어 메신저로 의견을 전달해야 했습니다.
UI 메모 남기기 기능은 이러한 부가적인 커뮤니케이션 비용을 줄이기 위해 고안된 기능입니다.

Example

// 단일 메모 쓰기

ecup.comment('#test_box', '테스트용 메모입니다.');

// 여러 메모를 한번에 달기

ecup.comment({

'#test_box1': '첫번째 테스트박스입니다.',

'#test_box2': '두번째 테스트박스입니다.',

'#test_box3': '세번째 테스트박스입니다.'

});

// 메모 화면에서 감추기

ecup.comment.show = false; // 감추기

ecup.layerControl( target, show, hide ) Layer DOM 제어하기
Options

Type: value or array

  • target , show , hide
    Type: String

    레이어 표시

    • target: 제어될 레이어 Dom

    • show: 레이어 활성화 이벤트 할당 Dom (ex. button)

    • hide: 레이어 비활성화 이벤트 할당 Dom (ex. button)

**
레이어를 보다 직관적이고, 쉽게 제어할 수 있는 기능입니다.

Example

// 단일 요소 제어하기

ecup.layerControl('.layer','.open_btn','close_btn');

// 여러 요소 제어하기

ecup.layerControl(['.layer','.dimmed'],['.open_btn','.profile_img'],['close_btn','.submit_btn']);

ecup.selectControl( button, list, { attribute } )selectbox(custom) 제어하기
Options

Type: String

  • button , list , attribute
    Type: String

    리스트 표시

    • button: 리스트 활성화 시킬 이벤트 할당 Dom

    • list: 활성화 될 리스트 Dom

    • attribute: 활성화와 함께 추가될 기타 속성 (ex. aria)

** aria 속성도 함께 제어
selectbox(custom)를 보다 직관적이고, 쉽게 제어 할 수 있는 기능입니다.
selectbox interaction 뿐만 아니라, aria-expanded, aria-selected, aria-hidden 과 같은 aria 속성도 함께 제어 됩니다.

Example

// option을 aria-select로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'aria-selected'});

// option을 클래스로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'.selected'});
