NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 마크업검수 레이어 # 코멘트 # UI 컴포넌트
ecup.markupLayer.set( { ... } )
Options

Type: Obeject

  • type (default: external)
    Type: string

    레이어 보기 방식

    • external: 디자인을 가리지 않게 창 내부에서 보기

    • internal: 레이어 형식으로 보기

    • windows: 새창 형식으로 보기

  • theme (default: dark)
    Type: string

    레이어 디자인 색상 테마

    • default, dark, naver

  • buttonType (default: check)
    Type: string

    버튼그룹의 동작방식 설정

    • check: 그룹내 버튼들이 체크박스처럼 토글로 동작

    • button: 그룹내 버튼들이 일반 버튼처럼 동작

    • radio: 그룹내 버튼들이 라디오처럼 동작

Example

ecup.markupLayer.set({

type: 'windows',

theme: 'dark',

buttonType: {

'그룹이름 1' : 'check' // 해당 그룹내 버튼타입 체크박스

'그룹이름 2' : 'radio' // 해당 그룹내 버튼타입 라디오

}

});

ecup.markupLayer( { name : event, ... } )
Arguments

Type: Obeject

  • name
    Type: String

    버튼에 표시될 내용

    • 버튼의 고유 값 (FK)

  • event
    Type: object or function

    버튼 이벤트

    • 버튼 이벤트 발생 시 동작할 내용

** Event
ecup 내에서 제공하는 옵션을 사용하거나,
직접 함수를 할당하는 2가지 방법으로 사용할 수 있습니다.

Example

// 버튼동작을 함수로 정의하기

ecup.markupLayer({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': function() {

// 버튼2 기능

}

});

// 사용자 함수시 토글 이벤트 할당하기

ecup.markupLayer({

'버튼1 이름': [

function() {

// on 기능

},

function() {

// off 기능

}

],

});

// 버튼동작을 옵션으로 정의하기

ecup.markupLayer({

'버튼1 이름': {

target: '#div', // 옵션을 적용시킬 타깃의 셀렉터,

addClass: 'class1 class2 class3' // 타깃에게 부여할 옵션 1 (옵션의 종류 보기)

toggleAttr: 'aria-label="닫기"' // 타깃에게 부여할 옵션 2 .. (여러개 가능)

},

'버튼2 이름': { ... }

});

ecup.markupLayer( name ).group( target, { name : event, ... } )
Arguments
  • name
    Type: String

    버튼 그룹명

    • 그룹을 식별하는 고유한 키값 (언어 무관)

Group Arguments

Type: Obeject

  • target
    Type: String

    공동 target 정의

  • name : event
    Type
    - key : string
    - value : object or function

    버튼 이벤트

    • name: 버튼 이름 (버튼의 고유 값)

    • event: 버튼 이벤트 발생 시 동작할 내용

** Group Options
▪ target : 그룹내의 버튼들의 여러 공동타깃을 한번에 정의 할 수 있습니다.
▪ nave : event : ecup.markupLayer( { . . . } )에서의 와 사용 방법이 동일합니다.

** 단일 버튼과 달리 그룹 버튼은 버튼의 타입을 지정할 수 있습니다.

Example

// 그룹화 하기

ecup.markupLayer('그룹 이름').group({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 이름': {

target: '#div',

addClass: 'class1 class2 class3'

}

});

// 이벤트 함수일때 그룹 DOM 사용하기 ($$ + 순서)

ecup.markupLayer('그룹 이름').group('#div', '.test', '#span', {

'버튼1 이름': function() {

$$1.addClass('good'); // #div에 적용

$$2.removeClass('test'); // .test에 적용

}

});

// 이벤트 옵션일때 그룹 DOM 사용하기

ecup.markupLayer('그룹 이름').group('#div', '.test', '#span', {

'버튼1 이름': {

hide: 400 // 그룹 DOM (#div, .test, #span) 에 적용됨

},

'버튼2 이름': {

target: '#wrap'

addClass: 'hello' // #wrap에 적용

}

});

Event Options
target

복수적용 : 공백

  • 이벤트가 할당될 셀렉터

    target: '#header'

클래스

복수적용 : 공백

  • 추가

    addClass: 'class1 class2'

    삭제

    removeClass: 'class1 class2'

    토글

    toggleClass: 'class1 class2'

속성

복수적용 : 공백 or 쉼표(,)

  • 추가

    addAttr: 'role="button" aria-label="닫기"'

    삭제

    removeAttr: 'role, aria-label'

    토글

    toggleAttr: 'role="button", aria-label="열기"'

인라인 스타일

복수적용 : 쉼표(,)

  • 추가

    addStyle: 'font-size: 13px; color #333;'

    삭제

    removeStyle: 'font-size, color'

    토글

    toggleStyle: 'width: 100%; background: red;'

Display

  • 보이기

    show: 400

    숨기기

    hide: 0

    토글

    toggle: 0

** 마크업 디자인 검수시 자주 사용되는 동작을 미리 정의해 놓았습니다.

ecup.comment( { target : comment } ) UI 코멘트 남기기
Arguments

Type: String, String

  • target
    Type: String

    코멘트 표시 할 요소 selector

  • comment
    Type: String

    코멘트 내용

Property
  • show
    Type: boolean (default: true)

    UI 코멘트 화면 표시 유무

    • true: 보이기

    • false: 숨기기

**
디자이너는 마크업개발자에게 제플린의 코멘트 기능을 활용해 기타 의견을 전달할 수 있지만,
마크업개발자는 디자이너에게 UI 검수를 받을 때 메모를 남길 수 없어 메신저로 의견을 전달해야 했습니다.
UI 메모 남기기 기능은 이러한 부가적인 커뮤니케이션 비용을 줄이기 위해 고안된 기능입니다.

Example

// 단일로 사용할 경우 아래와 같이 사용가능

ecup.comment('#test_box', '테스트용 메모입니다.');

// 단일 또는 다중 메모 남기기

ecup.comment({

'#test_box1': '첫번째 테스트박스입니다.',

'#test_box2': '두번째 테스트박스입니다.',

'#test_box3': '세번째 테스트박스입니다.'

});

// 메모 화면에서 감추기

ecup.comment.show = false; // 감추기

ecup.layerControl( target, show, hide ) Layer DOM 제어하기
Arguments

Type: value or array

  • target , show , hide
    Type: String

    레이어 표시

    • target: 제어될 요소 selector

    • show: 레이어 활성화 이벤트 할당 요소 selector (ex. button)

    • hide: 레이어 비활성화 이벤트 할당 요소 selector (ex. button)

** 요소 인자 개수
layerControl의 인자는 2개 또는 3개를 사용 할 수 있습니다.
2개 사용시 마지막 셀렉터는 활성화 시키는 요소입니다.

** 여러 요소 제어
여러개의 레이어 제어시 배열

Example

// 단일 요소 제어하기

ecup.layerControl('.layer','.open_btn','close_btn');

// 여러 요소 제어하기

ecup.layerControl(['.layer','.dimmed'],['.open_btn','.profile_img']);

ecup.selectControl( button, list, { attribute } )selectbox(custom) 제어하기
Arguments

Type: String

  • button
    Type: String

    리스트 활성화 시킬 이벤트 할당 요소 selector

  • list
    Type: String

    활성화 될 리스트 요소 selector

  • attribute
    Type: Objet

    활성화와 함께 추가될 기타 속성

    { '속성이 바뀔 요소 selector' : 'aria 혹은 class'}

** aria 속성도 함께 제어
selectbox(custom)를 보다 직관적이고, 쉽게 제어 할 수 있는 기능입니다.
selectbox interaction 뿐만 아니라, aria-expanded, aria-selected, aria-hidden 과 같은 aria 속성도 함께 제어 됩니다.

Example

// option을 aria-select로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'aria-selected'});

// option을 클래스로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'.selected'});
