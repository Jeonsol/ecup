NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 속성 # 인라인 스타일 # 스프라이트 # 애니메이션
$(selector).handleAttr( attributes ) 요소 속성 제어
Attributes

Type: String

속성값

여러 속성을 하나의 문자열 인자로도, 여러개의 문자열 인자로도 줄 수 있다. (예시 참조)

Example

** 속성 추가

$('#test').addAttr('role="button" aria-label="닫기"');

$('#test').addAttr('role="button"', 'aria-label="닫기"');

** 속성 삭제

$('#test').removeAttr('role, aria-label');

$('#test').removeAttr('role', 'aria-label');

** 속성 토글

$('#test').toggleAttr('role="button", aria-label="열기"');

$('#test').toggleAttr('role="button"', 'aria-label="열기"');

$(selector).handleStyle( styles ) 요소 인라인 스타일 제어
Styles

Type: String

스타일값

여러 스타일을 하나의 문자열 인자로도, 여러개의 문자열 인자로도 줄 수 있다. (예시 참조)

Example

** 인라인스타일 추가

$('#test').addStyle('background-color: #555; color: red;');

$('#test').addStyle('background-color: #555', 'color: red');

** 인라인스타일 삭제

$('#test').removeStyle('color, font-size');

$('#test').removeStyle('color', 'font-size');

** 인라인스타일 토글

$('#test').toggleStyle('width: 100%; height: 100%;');

$('#test').toggleStyle('width: 100%', 'height: 100%');

$(selector).sprite( options )스프라이트 이미지 gif 효과
Options

Type: String

  • url
    Type: String

    이미지 경로

    • 현재 요소에 적용된 이미지 외의 다른 이미지로 변경하고자 할 때 사용

  • count
    Type: Number ( default: 10 )

    스프라이트 프래임 개수

  • pos
    Type: Number ( default: 40 )

    프레임 하나의 이미지 크기

  • orient
    Type: String ( default: 'vertical')

    프레임 이동방향

    • vertical: background-position-x 변경

    • horizontal: background-position-y 변경

  • duration
    Type: Number ( default : 0.1 )

    프레임 전환 시간

    • 초단위

  • mode
    Type: String ( default: 'fix')

    애니메이션 효과 후 이미지 상태

    • fix: 애니메이션 동작 후 이미지 변경 상태 유지

    • remove: 애니메이션 동작 후 기본 상태로 돌아감

Example

$('.icon_select').sprite({

url: '../test.png',

count: 13,

pos: 40,

orient: 'vertical',

duration: 0.2,

mode: 'remove'

});

$.animation( name : '...', animate : { ... } )IE9 이하 애니메이션 - keyframe
Keyframes 정의
  • name
    Type: String

    키프레임 이름

  • animate
    Type: Object

    애니메이션으로 변경 될 속성

    • css keyframe 정의 방법과 유사

Example

$.animation({

name: 'layerUp',

animate: {

0: {

top: '100%',

},

100: {

top: '0',

}

},

});

$(selector).animation( keyFrameName , { options } ) IE9 이하 애니메이션 - keyframe 적용
KeyframeName
  • Type: String

    정의된 keyframe 이름

Options
  • duration
    Type: String or Number (default: 1000)

    애니메이션 동작 시간

  • delay
    Type: String or Number (default: 0)

    애니메이션 지연 시간

Example

$(selector).animation('layerUp', {

duration: '0.2s' or 200,

delay: '0.6s', or 600

});
