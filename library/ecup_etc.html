NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 임시 이미지
<img src="#" /> 임시이미지 자동적용 기능

사내에서 이미 임시이미지 기능이 구현된 웹사이트가 있습니다.

http://dev.ui.naver.com/tmp/generator.html 이 사이트로 여러 옵션을 선택해서 생성한 태그를 복사 해오시면 됩니다.

"이 조차도 귀찮다!" 하시는 분들을 위해 만든 기능으로

사내 임시이미지 사이트를 통해 동작하며, jQuery.js 와 Ecup.js가 추가되있으면 src="#" 만으로 임시 이미지를 불러옵니다.

다만 현재는 1:1 비율의 이미지만을 지원하며 주로 프로필 사진이나 썸네일 사진에서 활용하기 좋을 것이라 생각됩니다.

Example

// 랜덤 이미지 ( 인물, 라이프스타일, 배경 )

<img src="#" width="100" height="100" alt="임시이미지"/>

// 인물 랜덤 이미지

<img src="#p" width="100" height="100" alt="임시이미지"/>

<img src="#사람" width="100" height="100" alt="임시이미지"/>

// 라이프스타일 랜덤 이미지

<img src="#l" width="100" height="100" alt="임시이미지"/>

<img src="#라이프스타일" width="100" height="100" alt="임시이미지"/>

// 자연 랜덤 이미지

<img src="#n" width="100" height="100" alt="임시이미지"/>

<img src="#자연" width="100" height="100" alt="임시이미지"/>

// 고정 이미지 (뒤에 숫자 1 ~ 3)

<img src="#p1" width="100" height="100" alt="임시이미지"/>

<img src="#자연3" width="100" height="100" alt="임시이미지"/>
