Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 임시 이미지
ecup.comment() UI 메모 남기기
Options

Type: String, String

  • target : comment

    UI 코멘트 표시

    • target: 코멘트가 표시될 dom

    • comment: 코멘트 내용

Property
  • show
    Type: boolean

    UI 코멘트 화면 표시 유무

    • true: 보이기

    • false: 숨기기

**
디자이너는 마크업개발자에게 제플린의 코멘트 기능을 활용해 기타 의견을 전달할 수 있지만,
마크업개발자는 디자이너에게 UI 검수를 받을 때 메모를 남길 수 없어 메신저로 의견을 전달해야 했습니다.
UI 메모 남기기 기능은 이러한 부가적인 커뮤니케이션 비용을 줄이기 위해 고안된 기능입니다.

Example

// 단일 메모 쓰기

ecup.comment('#test_box', '테스트용 메모입니다.');

// 여러 메모를 한번에 달기

ecup.comment({

'#test_box1': '첫번째 테스트박스입니다.',

'#test_box2': '두번째 테스트박스입니다.',

'#test_box3': '세번째 테스트박스입니다.'

});

// 메모 화면에서 감추기

ecup.comment.show = false; // 감추기

<img src="#" /> 임시이미지 자동적용 기능

사내에서 이미 임시이미지 기능이 구현된 웹사이트가 있습니다.

http://dev.ui.naver.com/tmp/generator.html 이 사이트로 여러 옵션을 선택해서 생성한 태그를 복사 해오시면 됩니다.

"이 조차도 귀찮다!" 하시는 분들을 위해 만든 기능으로

사내 임시이미지 사이트를 통해 동작하며, jQuery.js 와 Ecup.js가 추가되있으면 src="#" 만으로 임시 이미지를 불러옵니다.

다만 현재는 1:1 비율의 이미지만을 지원하며 주로 프로필 사진이나 썸네일 사진에서 활용하기 좋을 것이라 생각됩니다.

Example

// 랜덤 이미지 ( 인물, 라이프스타일, 배경 )

<img src="#" width="100" height="100" alt="임시이미지"/>

// 인물 랜덤 이미지

<img src="#p" width="100" height="100" alt="임시이미지"/>

<img src="#사람" width="100" height="100" alt="임시이미지"/>

// 라이프스타일 랜덤 이미지

<img src="#l" width="100" height="100" alt="임시이미지"/>

<img src="#라이프스타일" width="100" height="100" alt="임시이미지"/>

// 자연 랜덤 이미지

<img src="#n" width="100" height="100" alt="임시이미지"/>

<img src="#자연" width="100" height="100" alt="임시이미지"/>

// 고정 이미지 (뒤에 숫자 1 ~ 3)

<img src="#p1" width="100" height="100" alt="임시이미지"/>

<img src="#자연3" width="100" height="100" alt="임시이미지"/>

ecup.layerControl() Layer DOM 제어하기
Options

Type: value or array

  • target , show , hide
    Type: String

    레이어 표시

    • target: 제어될 레이어 Dom

    • show: 레이어 활성화 이벤트 할당 Dom (ex. button)

    • hide: 레이어 비활성화 이벤트 할당 Dom (ex. button)

**
레이어를 보다 직관적이고, 쉽게 제어할 수 있는 기능입니다.

Example

// 단일 요소 제어하기

ecup.layerControl('.layer','.open_btn','close_btn');

// 여러 요소 제어하기

ecup.layerControl(['.layer','.dimmed'],['.open_btn','.profile_img'],['close_btn','.submit_btn']);

ecup.selectControl()selectbox(custom) 제어하기
Options

Type: String

  • button , list , attribute
    Type: String

    리스트 표시

    • button: 리스트 활성화 시킬 이벤트 할당 Dom

    • show: 활성화 될 리스트 Dom

    • hide: 활성화와 함께 추가될 기타 속성 (ex. aria)

** aria 속성도 함께 제어
selectbox(custom)를 보다 직관적이고, 쉽게 제어 할 수 있는 기능입니다.
selectbox interaction 뿐만 아니라, aria-expanded, aria-selected, aria-hidden 과 같은 aria 속성도 함께 제어 됩니다.

Example

// option을 aria-select로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'aria-selected'});

// option을 클래스로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'.selected'});

맨 위로 UI 주석 남기기 임시이미지 자동적용 기능 Layer DOM 제어하기 selectbox(custom) 제어하기
