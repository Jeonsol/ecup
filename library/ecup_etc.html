UI 메모 남기기

디자이너는 마크업개발자에게 제플린의 코멘트 기능을 활용해 의견를 달지만,

마크업개발자는 디자이너에게 UI 검수를 받을 때 메모를 남길 수 없어 메신저로 의견을 전달해야 했습니다.

UI 메모 남기기 기능은 이를 개선하고 커뮤니케이션 비용을 줄이기 위해 고안된 기능입니다.

사용방법은 다음과 같이 내가 코멘트를 남기고 싶은 DOM 요소와, 내용을 넣어주면 됩니다.

메모 토글버튼은 해당 DOM 요소의 좌측 상단에 위치하며 설정창에서 보이지 않게 설정할 수 있습니다.

// 단일 메모 쓰기

ecup.comment('#test_box', '테스트용 메모입니다.');

// 여러 메모를 한번에 달기

ecup.comment({

'#test_box1', '첫번째 테스트박스입니다.',

'#test_box2', '두번째 테스트박스입니다.',

'#test_box3', '세번째 테스트박스입니다.'

});

// 메모 화면에서 감추기

ecup.comment.show = false; // 감추기

임시이미지 자동적용 기능

사내에서 이미 임시이미지 기능이 구현된 웹사이트가 있습니다.

http://dev.ui.naver.com/tmp/generator.html 이 사이트로 여러 옵션을 선택해서 생성한 태그를 복사 해오시면 됩니다.

"이 조차도 귀찮다!" 하시는 분들을 위해 만든 기능으로

jQuery.js 와 Ecup.js가 추가되있으면 src="$.img" 만으로 임시 이미지를 불러옵니다.

다만 현재는 1:1 비율의 이미지만을 지원하며 주로 프로필 사진이나 썸네일 사진에서 활용하기 좋을 것이라 생각됩니다.

// 랜덤 이미지 ( 인물, 라이프스타일, 배경 )

<img src="$.img" width="100" height="100" alt="임시이미지"/>

// 인물 이미지

<img src="$.img1" width="100" height="100" alt="임시이미지"/>

// 라이프스타일 이미지

<img src="$.img2" width="100" height="100" alt="임시이미지"/>

// 배경 이미지

<img src="$.img3" width="100" height="100" alt="임시이미지"/>

Layer DOM 제어하기

Layer를 보다 직관적이고, 쉽게 제어할 수 있는 기능입니다.

사용 방법은 다음과 같이 제어하고 싶은 Layer Dom 요소와, Layer Dom 요소를 보이게 하는 버튼 요소와, Layer Dom 요소를 가리게 하는 버튼 요소를 넣어주면 됩니다.

// 단일 요소 제어하기

ecup.layerControl('.layer','.open_btn','close_btn');

// 여러 요소 제어하기

ecup.layerControl(['.layer','.dimmed'],['.open_btn','.profile_img'],['close_btn','.submit_btn']);

selectbox(custom) 제어하기

selectbox(custom)를 보다 직관적이고, 쉽게 제어 할 수 있는 기능입니다.

selectbox interaction 뿐만 아니라, aria-expanded, aria-selected, aria-hidden 과 같은 aria 속성도 함께 제어 됩니다.

사용 방법은 다음과 같이 버튼 요소와, listbox 요소와, option 요소를 넣어주면 됩니다.

// option을 aria-select로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'aria-selected'});

// option을 클래스로 제어하기

ecup.selectControl('.button','.listbox',{'.option':'.selected'});
