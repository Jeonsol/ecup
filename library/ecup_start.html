NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup ( Easy Control Markup ) jQuery 확장 메서드 부가 기능
# 마크업검수 레이어
Browser support

IE 8+ (possibly 9 also), latest of Chrome/FF/Safari, iOS 7+ and Android 2.1+ (except 3.x)


Load files or import library

src="http://view.gitlab2.uit.navercorp.com/NT11398/ecup/raw/develop/js/ecup.js"


Initialize

// 레이어 환경 설정 ( ex. 노출 타입, 디자인 테마 )

ecup.markupLayer.set({

type: 'external',

theme: 'dark'

external: {

btnText: '마크업',

right: 50,

bottom: 50

},

buttonType: {

'로그인 상태': 'radio'

}

});

// 버튼 생성 ( 단일 or 그룹 )

ecup.markupLayer({

'버튼1 이름': function() {

// 버튼1 기능

},

'버튼2 기능': function() {

// 버튼2 기능

}

});
