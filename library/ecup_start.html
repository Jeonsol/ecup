NTS 마크업 라이브러리

라이브러리 소개

Ecup.js

Easy Control markUP

마크업개발자가 디자인검수를 위해 임시로 구현하는 HTML, JS 의 개발 리소스를 줄여주는 자바스크립트 라이브러리 입니다.

크로스브라우징 이슈를 최소화하기 위해서 jQuery 에 의존성을 가지기 때문에, jQuery 라이브러리를 먼저 불러오신 다음 사용해주시기 바랍니다.

시작하기 Ecup jQuery 확장 메서드 부가 기능
Browser support

IE 8+ (possibly 9 also), latest of Chrome/FF/Safari, iOS 7+ and Android 2.1+ (except 3.x)


Load files or import library

<script src="http://view.gitlab2.uit.navercorp.com/NT11398/ecup/raw/develop/js/ecup.js"/>


Initialize

// 레이어 환경 설정 ( ex. 노출 타입, 디자인 테마 )

ecup.markupLayer.set({

type: 'external',

theme: 'dark'

external: {

btnText: '마크업',

right: 50,

bottom: 50

},

buttonType: {

'로그인 상태': 'radio'

}

});

검수 레이어 버튼 추가 및 활성화
단일 버튼

ecup.markupLayer({

'로딩': {

target: '.list_article',

toggleClass: 'is_loading',

},

});

그룹 버튼

ecup.markupLayer('로그인 상태').group({

'로그인 전': function(){

$(".header .button_login").show();

$(".header .area_user").hide();

},

'로그인 후': function(){

$(".header .button_login").hide();

$(".header .area_user").show();

}

});

마크업 가이드 레이어 활성화

ecup.markupLayer.show();

UI 코멘트 달기
단일 코멘트

ecup.comment('.content_image','콘텐츠 이미지 영역입니다.');

레이어 제어하기
단일 추가 제어

ecup.layerControl('.layer','.open_btn','close_btn');

다중 추가 제어

ecup.layerControl(['.layer','.event_btn'],['.profile_img','.open_btn']);

셀렉트박스 제어하기
.option을 aria-select로 제어

ecup.selectControl('.button','.listbox',{'.option':'aria-selected'});

요소 속성 제어

$('#test').$addAttr('role', 'button');

$('#test').$removeAttr('role', 'aria-label');

$('#test').$toggleAttr({

role: 'button',

'aria-label': '테스트'

});

요소 인라인 스타일 제어

$('#test').$addStyle('background-color', '#333');

$('#test').$removeStyle('background-color, color');

$('#test').$toggleStyle({

display: 'block',

width: '100px'

});

스프라이트 이미지 애니메이션 효과

$('.icon_select').sprite({

count: 13, // 프레임 개수

pos: 40, // 프레임 사이즈

orient: 'vertical', // 이동 방향

duration: 0.2, // 동작 시간

mode: 'remove' // 효과 종료 후 스타일 유무

});
